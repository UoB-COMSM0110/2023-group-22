## For Development
1. Create branch and start develop.
```
git checkout -b <branch_name>
```

1. check you are on your own branch
```
git branch
```

1. commit you change
```
git add .
git commit -m <message>
```

1. push the commit on the branch
```
git push -u origin <branch_name>
```
